
# Parser News Bot

This bot can parse the news from the site kun.uz :)


## Installation

Install my-project
```bash
git clone https://github.com/mython-dev/Parser-news.git
```

## Usage/Examples

```python3
cd Parser-news
pip3 install -r requirements.txt
nano config.py # add your token:)
python3 bot.py
```
